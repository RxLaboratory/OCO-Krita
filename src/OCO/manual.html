OCO - Open Cut-Out exporter for Krita

OCO is an open format to ease the exchange of 2D cut-out assets between different applications.

To Export to the OCO format, go to Tools ‣ Scripts ‣ OCO Export.

To learn more about OCO, visit rxlaboratory.org.

To learn how to use the OCO exporter for Krita and what it can do, visit oco-krita.rxlab.guide.
